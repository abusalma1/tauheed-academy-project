JavaScript Usage

Copy these functions into your projects to use notifications:

// Success notification
showSuccessNotification('Message here');

// Error notification
showErrorNotification('Error message here');

// Warning notification
showWarningNotification('Warning message here');

// Info notification
showInfoNotification('Info message here');

// Loading notification (duration in seconds)
showLoadingNotification('Loading...', 5);
